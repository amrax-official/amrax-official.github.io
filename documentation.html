Hybrid Market Condition (HMC)

The core logic that detects trend strength using ADX to filter bad market conditions.

HMC_Enable Activates the trend-filtering logic.
HMC_ADX_Threshold Default 25.0. Higher means stronger trend required.
HMC_HedgeLotMult Lot multiplier for recovery hedging.

Intelligent Grid Management

Grid_Mode ATR Recommended for volatility-based spacing.
PGS_Enable Progressive spacing to prevent heavy drawdown in streaks.
DynGrid_Cooldown Blocks new grid layers during extreme volatility.

Partial Close Losing (PCL)

Institutional technique to "shave off" bad trades using current profits.

PCL_TriggerUSD Starts PCL after reaching this profit amount.
PCL_UsePct Uses 80% of current profit to cancel out losing trades.

Optimized Trading Windows

AMRAX supports 4 customizable sessions. Default is set to Smart Session End mode.

Sess_EndMode Smart Mode closes cycles safely before session ends.

Daily Targets & Protections

Risk_DD_Pct Hard stop if account reaches X% drawdown.
Risk_ProfitPct Daily profit target (e.g., 5%). Stop for the day once hit.
